# GUIXmlChange

## GUI 版本

讀取 .INI 檔 轉換成 xml

檔案中未含 INI, 請自行加入

## 使用方法

引數輸入

input  argument (.INI): /i
output argument (.xml): /o
exit   argument : /q

EX : 
1. /i DNWEB.INI /o END.xml 
2. /q
